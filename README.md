# Minicar Workshop

PiCar-Xミニカーのプログラミングワークショップ教材です。

## 学習教材

ワークショップの詳細な学習教材はこちら:

[https://tinyurl.com/minicar-workshop](https://tinyurl.com/minicar-workshop)

## プログラム一覧

### 基本操作 (00-04)
- `00_move_head.py` - カメラのパン/チルト制御
- `01_move_forward.py` - 前進
- `02_move_backward.py` - 後退
- `03_turn_left.py` - 左折
- `04_turn_right.py` - 右折

### センサーと条件分岐 (05-08)
- `05_measure_once.py` - 超音波センサーの使い方
- `06_if_statement.py` - if文
- `07_if_else.py` - if/else文
- `08_elif_distance.py` - elifと距離センサー

### 応用ロジック (09-11)
- `09_add_backward.py` - 後退機能の追加
- `10_while_elif.py` - whileループとelif
- `11_try_finally.py` - 例外処理

### センサー統合 (12-14)
- `12_use_power.py` - パワー制御
- `13_use_distance.py` - 距離ベースのナビゲーション
- `14_use_angle.py` - 角度ベースの操舵

### 最終チャレンジ (15)
- `15_challenge_tuning.py` - 障害物回避システム（パラメータ調整可能）

## 使い方

各Pythonファイルを順番に実行して、ミニカーのプログラミングを学習します。

```bash
python3 00_move_head.py
```
